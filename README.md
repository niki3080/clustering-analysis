# Customer Segmentation Clustering

Проект содержит исследование клиентской сегментации с помощью кластерного анализа на датасете `Customer Personality Analysis`.

## Содержимое

- `3_ml_homework_clustering.ipynb` — основной ноутбук с анализом, визуализацией, PCA и кластеризацией.
- `data/customer_segmentation.csv` — исходный датасет, используемый в ноутбуке.
- `styles/` — дополнительные визуальные настройки и стили для графиков.
- `process/` — вспомогательные и архивные файлы, которые не нужны для основной публикации.

## О проекте

Задача: проанализировать клиентскую базу и выделить сегменты на основе потребительского поведения. Основные этапы:

- загрузка и предварительное исследование данных
- работа с пропущенными значениями и выбросами
- анализ распределений и покупательских паттернов
- преобразование и масштабирование признаков
- кластеризация клиентов
- оценка качества сегментации с помощью силуэта и других метрик

## Как запустить

1. Откройте `3_ml_homework_clustering.ipynb` в Jupyter Notebook или JupyterLab.
2. Выполните ячейки последовательно.
3. Убедитесь, что файл `data/customer_segmentation.csv` доступен в папке `data/`.

Если файл недоступен, ноутбук попытается загрузить его из резервного Dropbox URL.

## Примечания

- `process/` содержит дополнительные файлы и экспортированные материалы, которые не включены в основное исследование.
- `data/` — единственная директория с данными, необходимая для воспроизведения анализа.
