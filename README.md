# ML Homework Projects

Проект содержит две работы: исследование клиентской сегментации с помощью кластерного анализа и дизайн A/B-теста.

## Содержимое

- `3_ml_homework_clustering.ipynb` — основной ноутбук с анализом, визуализацией, PCA и кластеризацией.
- `4_ml_AB.ipynb` — ноутбук с расчетами и анализом для A/B-теста.
- `AB_test_design.pdf` — документ с дизайном A/B-теста.
- `data/customer_segmentation.csv` — исходный датасет для ноутбука по кластеризации.
- `data/ab_test_dataset.csv` — датасет для ноутбука по A/B-тесту.

## Кластеризация

Задача: проанализировать клиентскую базу и выделить сегменты на основе потребительского поведения. Основные этапы:

- загрузка и предварительное исследование данных
- работа с пропущенными значениями и выбросами
- анализ распределений и покупательских паттернов
- преобразование и масштабирование признаков
- кластеризация клиентов
- оценка качества сегментации с помощью силуэта и других метрик

## A/B-тест

Задача: подготовить и проанализировать дизайн A/B-теста. Основные материалы:

- ноутбук с расчетами по эксперименту
- PDF-документ с описанием дизайна теста

## Как запустить

1. Откройте нужный ноутбук в Jupyter Notebook или JupyterLab.
2. Выполните ячейки последовательно.
3. Убедитесь, что нужный CSV доступен в папке `data/`.

Если `data/customer_segmentation.csv` недоступен, ноутбук по кластеризации попытается загрузить его из резервного Dropbox URL.

## Примечания

- `data/` — единственная директория с данными, необходимыми для воспроизведения опубликованных ноутбуков.
